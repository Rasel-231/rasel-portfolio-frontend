# 🚀 Rasel Portfolio

A modern and high-performance full stack developer portfolio built with Next.js, TypeScript, Tailwind CSS, Node.js, and MongoDB to showcase projects, skills, and professional experience.

## 📦 Installation

```bash
git clone https://github.com/yourusername/rasel-portfolio.git

cd rasel-portfolio

npm install

npm run dev
```

---

## 🔐 Environment Variables

Create a `.env.local` file and add the following:

```env
NEXT_PUBLIC_API_URL=
MONGODB_URI=
JWT_SECRET=
```
